
TechMaster Brasil - Site pronto para Netlify
===========================================

Conteúdo:
- index.html
- styles.css
- articles.html
- about.html
- contact.html
- /articles/*.html (8 artigos)

Como publicar no Netlify:
1. Faça login no Netlify e crie um novo site a partir de 'Deploy site' -> 'Deploy manually'.
2. Arraste o conteúdo da pasta para o deploy (index.html, styles.css, articles/).
3. Verifique se seu domínio está configurado corretamente (opcional).

Como adicionar AdSense (instruções rápidas):
1. Não adicione o código AdSense até que o conteúdo esteja no site e público.
2. Adicione o script fornecido pelo Google no &lt;head&gt; do index.html, e blocos de anúncio nas páginas que desejar.
3. Aguarde o Google revisar seu site e, se necessário, corrija itens solicitados.
4. Recomendações para aprovação: mantenha conteúdo original, meta descriptions, página 'About', 'Contact' e políticas (privacy policy).

Observações:
- Substitua o e-mail de contato em contact.html por seu e-mail real.
- Se desejar, posso separar os arquivos em um repositório Git ou configurar Netlify _redirects / headers_.
